O arquivo "gerador_movel.ino" contém o código enviado ao ESP32 associado ao sistema de geração com rastreamento. O código contém a aquisição e envio de dados, bem como a lógica de movimentação do protótipo.
O arquivo "gerador_fixo.ino" contém o código enviado ao ESP32 associado ao sistema de geração fixo. O código contém a aquisição e envio de dados.
